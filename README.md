# dashboard_de_vendas
Dashboard de vendas
Xbox Game Pass – Sales Dashboard (Excel)

Projeto desenvolvido com o objetivo de criar um Dashboard de Vendas no Excel para o Lab Project da DIO, transformando dados brutos em informações visuais claras e estratégicas para análise de desempenho.

Sobre o Projeto:

O dashboard apresenta uma visão consolidada das vendas de assinaturas relacionadas ao Xbox Game Pass, permitindo acompanhar resultados de forma visual, organizada e interativa.

Funcionalidades do Dashboard

✅ Total de vendas por produto (EA Play Season Pass e Minecraft Season Pass)

✅ Total geral de assinaturas

✅ Filtro por tipo de assinatura (Anual, Mensal e Trimestral)

✅ Visualização gráfica comparativa

✅ Layout intuitivo e organizado

Dados Utilizados

A base de dados contém informações relacionadas a:

Tipo de assinatura

Produto

Valores de venda

Período de análise

Os dados foram tratados e organizados para permitir consolidação automática e atualização dinâmica do dashboard.
